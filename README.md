# 家族目標共有システム 要件定義書

## 1. プロジェクト概要

### システム名（仮称）

**Family Goals（家族目標共有システム）**

### プロジェクト目的

家族間で目標やタスクを共有し、お互いに応援・見守ることで継続率を向上させる。

「目標を宣言すると達成しやすくなる」という心理効果（コミットメント効果）を利用し、一人では続かない習慣を家族全員で支え合うことを目的とする。

また、本システムはAIエージェントを活用したチーム開発を学習するための研修課題として開発する。

# 3. システム概要

家族単位で利用するWebサービス。

利用者は

* 家族を作成
* 目標を宣言
* 毎日の達成報告
* 家族から応援
* ご褒美設定

を行う。

システムは達成率や連続記録などを可視化する。

# 15. 学習目標

本プロジェクトでは、単にアプリケーションを完成させることではなく、以下のスキルを身につけることを目的とする。

* 要件定義・設計書作成
* GitHubによるドキュメント管理
* AIエージェントへの適切なタスク分割
* Pull Requestレビュー
* Dockerを利用した開発環境構築
* AWSへのデプロイ
* フロントエンド・バックエンド・AI分析の連携
* 保守性・拡張性を考慮した設計

これらを通じて、「AIと協働するソフトウェア開発」の実践的な経験を積むことを最終目標とする。
